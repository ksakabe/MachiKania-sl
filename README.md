# KM-BASIC版 `sl`

`SL.BAS`は、Toyoda Masashi氏の`sl` 5.03(https://github.com/mtoyoda/sl )をKM-BASIC向けに移植したものです。D51、C51、LONG編成が、煙と6コマの動輪アニメーションを伴って画面右端から左端へ走ります。

## 実行

1. `SL.BAS`をMachiKaniaで読み込みます。
2. [公式STRDIMクラス](https://github.com/machikania/lib/tree/master/LIB/STRDIM)の`STRDIM.BAS`を、メインプログラムと同じディレクトリ、またはSDカードの`\LIB\STRDIM\STRDIM.BAS`へ配置します。
3. エディタ画面でF4（RUN）を押します。
4. 強制停止する場合は、機種ごとの通常のBASIC停止操作を使います。

画面サイズは`SYSTEM(20)`と`SYSTEM(21)`で取得するため、30桁・40桁・42桁・80桁などのテキストモードに追従します。列車の画面外部分は自動的にクリップされます。

列車本体、動輪、炭水車の文字列配列には、公式`STRDIM`クラスの`SETSTR`および`GETSTR$`メソッドを使用しています。
C51の炭水車は機関車より1行低い12行構成のため、D51とは別の専用データとして保持しています。

## 実行中の切り替え

- `↑`（UP）: 現在選択中の列車を飛行させる
- `→`（RIGHT）: C51へ切り替える（通常走行）
- `↓`（DOWN）: D51へ切り替える（通常走行）
- `←`（LEFT）: LONG編成へ切り替える（通常走行）

## 元プログラムとの差

KM-BASICには通常のコマンドライン引数がないため、`-F`、`-l`、`-c`相当の機能は実行中の方向キーで選択します。`-a`相当は含みません。描画は互換性を優先してフレームごとに`CLS`を行います。
